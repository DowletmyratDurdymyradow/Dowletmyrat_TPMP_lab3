# Overview

Source C project with structures.

# Usage

Use `make` to build project.

# Building

Use follow command:

```bash
make
```

Run app:

```bash
make run
```

# Database initialization

Create local database from SQL script:

```bash
cd src
sqlite3 students.db < init_students.sql
cd ..
```
