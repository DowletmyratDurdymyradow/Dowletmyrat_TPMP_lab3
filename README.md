# Overview

Source C project for sqlite database using.

# Usage

Use `make`, `Makefile` and `sqlite3` to build project.

# Building

Use follow command:

```bash
make
```

# Database initialization

Create local database from SQL script:

```bash
cd src
sqlite3 students.db < init_students.sql
cd ..
```

# Run examples

```bash
./example1
./example2
./example3
./students
```
